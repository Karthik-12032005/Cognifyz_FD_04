# Smart Authentication Form

Smart Authentication Form is a modern and responsive authentication system developed using HTML, CSS, and JavaScript. The project demonstrates advanced form validation, dynamic DOM manipulation, password strength checking, and interactive UI design.

---

## 🚀 Features

- User Registration Form
- Password Strength Checker
- Confirm Password Validation
- Show/Hide Password Feature
- Real-Time Error Messages
- Dynamic DOM Manipulation
- Responsive Modern UI
- Smooth CSS Animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
